### 인증 스코프
- **PUBLIC**: 인증 불필요
- **TRUSTED_DEVICE**: 관리자 승인을 받은 기기 신뢰 토큰 필요
- **TRACK**: 트랙 세션 토큰 필요 (진행자가 PIN 로그인 후 발급)
- **ADMIN**: 관리자 액세스 토큰 필요
- **ADMIN_REFRESH**: 관리자 리프레시 토큰 전용 (액세스 토큰 재발급 엔드포인트에만 사용)

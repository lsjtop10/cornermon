## 기존 코드 이해

- `group_list_screen.dart`는 필터 행과 표 카드가 서로 다른 여백 컨테이너에 있어 코너·트랙 관리 화면의 단일 `16px` 콘텐츠 컬럼과 기준선이 맞지 않았다.
- `track_bulk_manage_screen.dart`는 필터와 표를 하나의 `Padding(16)` + `crossAxisAlignment: CrossAxisAlignment.stretch` 컬럼에 둔다.

## 확인 결과

- `AdminScaffold`는 관리자 캠프 내부 라우트의 공용 셸이며, 현재 최상단 `Row`에 `SafeArea`가 없다.
- Flutter 공식 SafeArea 지침에 따라 시스템 inset은 `SafeArea`로 처리하고, UI 간격은 별도 8dp 패딩으로 둔다.
